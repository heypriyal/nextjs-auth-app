# Next.js Authentication App

This is a simple Next.js project demonstrating user authentication, protected routes, and product listing.

## Features
- User Login
- Protected Home Page
- Logout Functionality
- Product Listing from API

## Tech Stack
- Next.js (React Framework)
- React
- Tailwind CSS
